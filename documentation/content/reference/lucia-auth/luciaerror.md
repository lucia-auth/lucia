All [known errors](/basics/error-handling#known-errors) are thrown as `LuciaError`, which is a standard `Error` :

```ts
class LuciaError extends Error {}
```

## properties

### `message`

The error message.

```ts
const error: LuciaError;
const errorMessage = error.message;
```
